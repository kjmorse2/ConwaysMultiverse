# Conways Multiverse
## Author: Kenneth Morse
### Inspired by by carykh on YouTube
See his video [here](https://www.youtube.com/watch?v=QK_KZv-YyOc)

### Features:
- Step through Conways Game of life.
- Live edit the grid to add/subtract cells by clicking on the cell you would like to change.
